# Latest Updates (2026/04/19)

Tested with ESPHome 2026.4.0.

Changes:

1. Fixed the `wifi.yaml` file by removing the dynamic icon. It appears that this function is no longer available at this time. See [Icon and device class getter migration](https://developers.esphome.io/blog/2026/03/12/icon-and-device-class-getter-migration/) for more information.
2. Added new binary sensors:
   - TTP223 capacitive touch button.
   - AHT20 or AHT30 temperature and humidity sensor.
   - ENS160 volatile organic compound (VOC) and eCO2 sensor.
3. Added new Bluetooth sensors / integrations:
   - Xiaomi Mi Smart Scale 2 Bluetooth sensors.
   - Mopeka Pro Check BLE sensor.
