# Changelog (2026/05/11)

Initial ESPHome Modular Rebuild 2026 setup and cleanup.

## Breaking Changes

The repository now uses lowercase paths with underscores for module folders and
module filenames. Any existing device YAML or package includes that referenced
the old copied paths must be updated.

Renamed paths include:

- `sensors/Bluetooth/` -> `sensors/bluetooth/`
- `sample-projects/` -> `sample_projects/`
- `documentation/deferred-components/` -> `documentation/deferred_components/`
- `documentation/project-structure.md` -> `documentation/project_structure.md`
- `sample-secrets.yaml` -> `sample_secrets.yaml`
- `includes/picow_intLED.h` -> `includes/picow_int_led.h`
- `boards/esp32/c3-oled-0-42-display.yaml` -> `boards/esp32/c3_oled_0_42_display.yaml`
- `sensors/binary/hc-sr501.yaml` -> `sensors/binary/hc_sr501.yaml`
