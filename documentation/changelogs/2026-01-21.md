# Latest Updates (2026/01/21)

Last tested with ESPHome 2025.12.

2. ESPNow has been added to the network folder, but it has not been tested yet.

4. WiFi module updated to cater for auth changes in ESPHome 2025.11 and roaming in ESPHome 2026.1.
5. SPS30 sensor module: added AQI for EU and US, and added device classes for AQI and PM.
6. Work in progress: added device classes to most sensors, plus other minor tweaks.
7. Removed the following files until calibration is confirmed:

   - `sparkfun_rain_gauge.yaml`
   - `sparkfun-wind_vane.yaml`
8. Most boards have been moved to ESP-IDF and tested.
9. Work in progress: file and folder cleanup is continuing.
10. Work in progress: adding further documentation to all files is ongoing.
